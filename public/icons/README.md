# PWA Icons

This directory should contain the PWA icons:

- `icon-192.png` - 192x192 pixel icon
- `icon-512.png` - 512x512 pixel icon

## Generate Icons

You can generate the icons using:

1. **Online tools**:
   - https://www.pwabuilder.com/imageGenerator
   - https://realfavicongenerator.net/
   - https://favicon.io/

2. **Local tools**:
   - ImageMagick: `convert input.png -resize 192x192 icon-192.png`
   - GIMP, Photoshop, etc.

## Suggested Theme

The icons should represent green plants/leaves, aligned with the application theme.
